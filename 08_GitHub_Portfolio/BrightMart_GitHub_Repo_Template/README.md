# BrightMart Retail Analytics

## Project Overview
This project analyzes BrightMart retail sales data to identify performance patterns and provide recommendations for improving profitability.

## Business Problem
BrightMart has experienced increasing sales but inconsistent profitability. Management wants to understand regional, category, customer-segment, and discount-related performance.
